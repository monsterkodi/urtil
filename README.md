# urtil
### makes url tiles

```sh

npm install -g urtils
echo "github.com" >> tiles
echo "npmjs.com" >> tiles
urtil tiles
open tiles.html

```

The result looks like this:

![tiles](http://monsterkodi.github.io/urtil/img/tiles.png)
