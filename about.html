urtil

aboutexample

urtil is a little tool that generates html pages, similar to the "New Tab" page of Firefox.

When you provide it with a list of webpages like this: 

github.com
npmjs.com

It will generate an html page that looks like this:

example configs

Some notes on the config files from the main example:

example

config
    tileSize    300
tiles
useless
    break
monsterkodi
    break

The first two lines tell urtil to make 300 pixel square tiles.

If a top level entry contains no dot, urtil assumes it is a local html file.

The break keyword centers the preceeding tiles and lets the next tile begin on a new line.

monsterkodi

config
    tileSize      300
monsterkodi.net
    image         ../monsterkodi.net.png
monsterkodi.github.io
    break
    
github.com/monsterkodi
npmjs.com/~monsterkodi
    break
    
projects
    screenHeight  960
    refresh
    break

The image keyword (4th line) can be used to provide an image in case the website doesn't render with the underlying webshot.

If urtil has created a thumbnail image for a webpage in previous runs, it doesn't regenerate it by default. The refresh keyword forces it to regenerate the thumbnail on every run.
